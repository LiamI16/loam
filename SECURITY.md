# Security Policy

> ⚠️ Placeholder — this is an early-stage project.

## Scope

Loam is an **offline, on-device** music engine. It runs entirely in the
user's browser/app: no server, no backend, no network calls, no user
accounts, and no stored personal data. As such, the security surface is
limited primarily to:

- the safety of published npm packages and their dependencies, and
- robustness against malformed input (e.g. crafted seed strings).

## Reporting a vulnerability

If you find a security issue, please report it privately rather than
opening a public issue:

- Use GitHub's **"Report a vulnerability"** feature under the
  [Security tab](https://github.com/LiamI16/loam/security), or
- Open a regular issue for non-sensitive bugs.

We'll acknowledge reports as soon as we reasonably can. Since this is a
small project, please be patient with response times.

## Supported versions

Loam is pre-1.0 (`0.0.x`). Only the latest `main` is supported.
