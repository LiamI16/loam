# Obsidian Plugin Brainstorming

This note is intended as a brainstorming file for features that are specific to the obsidian plugin that relies on the loam engine.

## Per Note Track with Smooth Transitions

When taking notes and gathering information, we learn and associate ideas better when using more senses. If we assoicate a different track to each note (the title of the note can be a seed) the brain can internally differentiate between two different ideas. When switching from one tab to another, there must be a smooth, slow transition to make this unnoticable.

## Pomodoro Timer / Study Timer / (Sleep Alarm?)
Built in timer with an alarm or special track to notify user of user-set time periods.
